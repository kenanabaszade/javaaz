Java

Oracle Java bir nömrəli proqramlaşdırma dili və inkişaf platformasıdır. O, xərcləri azaldır, inkişaf müddətlərini qısaldır, innovasiyanı təşviq edir və tətbiq xidmətlərini yaxşılaşdırır. Dünya üzrə milyonlarla developer tərəfindən 60 milyarddan çox Java Virtual Machine (Java Virtual Maşın) işlətdiyi üçün Java həm də müəssisələr və developerler üçün üstünlük verilən inkişaf platforması olaraq qalmaqdadır.

Java Proqramlaşdırma Dilinin Xüsusiyyətləri

Java, 1995-ci ildə Sun Microsystems (indiki Oracle) tərəfindən buraxılmış, obyekt-orientasiyalı, platformadan asılı olmayan və geniş istifadə sahəsinə malik bir proqramlaşdırma dilidir. Java-nın populyarlığı onun çoxsaylı xüsusiyyətləri ilə bağlıdır. Bu xüsusiyyətlər aşağıda ətraflı təsvir edilmişdir.

  1. 1. Platformadan Asılı Olmaması

    Java-nın ən mühüm xüsusiyyətlərindən biri onun platformadan asılı olmamasıdır. "Bir dəfə yaz, hər yerdə çalıştır" (Write Once, Run Anywhere - WORA) prinsipi Java-nın əsasını təşkil edir. Java proqramları Java Virtual Machine (JVM) üzərində çalışır. Bu, proqramçılara eyni kodu fərqli platformalarda yenidən yazmadan istifadə etməyə imkan verir.

  2. 2. Obyekt-Orientasiyalı Proqramlaşdırma

    Java, obyekt-orientasiyalı proqramlaşdırma (OOP) prinsiplərinə əsaslanır. Bu, kodun modullu, təkrar istifadə edilə bilən və asan idarə olunan olmasını təmin edir.

  3. 3. Sadəlik

    Java, C və C++ kimi dillərdən ilhamlanaraq yaradılmışdır, lakin daha sadə sintaksisə malikdir. Java, C++-dakı bəzi mürəkkəb xüsusiyyətləri (məsələn, pointerlər və çoxsaylı miras alma) aradan qaldıraraq proqramçıların işini asanlaşdırır. Bu, yeni başlayanlar üçün öyrənməni daha əlçatan edir.

  4. 4. Təhlükəsizlik

    Java, təhlükəsizlik baxımından güclü bir dildir. Java-nın təhlükəsizlik mexanizmləri aşağıdakılardan ibarətdir:

    • Bytecode Doğrulaması: JVM, proqramın icrasından əvvəl bytecode-u yoxlayır və potensial təhlükəli əməliyyatları bloklayır.
    • Sandbox Modeli: Java apletləri məhdudlaşdırılmış bir mühitdə (sandbox) işləyir, bu da zərərli kodun sistemə zərər verməsinin qarşısını alır.
    • Təhlükəsizlik Meneceri: Java-nın Security Manager sinfi proqramların icazəsiz əməliyyatlar yerinə yetirməsini məhdudlaşdırır.
  5. 5. Çoxaxınlılıq (Multithreading)

    Java, çoxaxınlılıq (multithreading) dəstəyi ilə proqramların eyni anda bir neçə tapşırığı yerinə yetirməsinə imkan verir. Bu xüsusiyyət, xüsusilə server tətbiqləri və real vaxt sistemləri üçün faydalıdır. Java-nın Thread sinfi və Runnable interfeysi çoxaxınlı proqramlaşdırmanı asanlaşdırır.

  6. 6. Zəngin API və Kitabxanalar

    Java, geniş və funksional API-lara malikdir. Java Standard Kitabxanası (Java Standard Library) şəbəkə əməliyyatları, fayl idarəetməsi, verilənlər bazası ilə iş, qrafik interfeyslər və s. kimi bir çox sahədə funksionallıq təmin edir. Bundan əlavə, Java-nın geniş ekosistemi çoxsaylı üçüncü tərəf kitabxanalarını və framework-ləri (Spring, Hibernate və s.) dəstəkləyir.

  7. 7. Zibil Toplayıcı (Garbage Collection)

    Java, avtomatik yaddaş idarəetməsi ilə tanınır. Zibil toplayıcı (Garbage Collector) artıq istifadə olunmayan obyektləri yaddaşdan təmizləyir. Bu, yaddaş sızması (memory leak) problemlərini minimuma endirir və proqramçıların yaddaş idarəetməsi ilə bağlı narahatlıqlarını azaldır.

  8. 8. Geniş İstifadə Sahəsi

    Java, müxtəlif sahələrdə istifadə olunur:

    • Veb tətbiqləri: Spring və JavaServer Faces kimi framework-lər ilə.
    • Mobil tətbiqlər: Android proqramlaşdırma üçün Java və Kotlin əsas dillərdir.
    • Müəssisə tətbiqləri: Java EE platforması böyük miqyaslı biznes tətbiqləri üçün istifadə olunur.
    • Oyun inkişafı: LibGDX kimi kitabxanalarla oyun yaradılır.
    • Böyük məlumat və süni intellekt: Hadoop və Spark kimi alətlər Java əsasında işləyir.

Java Kodlarına Nümunələr

1. Əsas "Hello World" Başlanğıc üçün
public class Main {
    public static void main(String[] args) {
        System.out.println("Salam, Dünya!"); // Konsola çıxış
    }
}
Məqsəd: Java'nın ən əsas sintaksisini nümayiş etdirir - class, metod və çıxış əmri.
2. Dəyişənlər və Şərt Operatoru (Fundamental konseptlər)
public class Weather {
    public static void main(String[] args) {
        int temperature = 23; // Dərəcə
        if (temperature > 30) {
            System.out.println("Hava çox istidir");
        } else if (temperature > 20) {
            System.out.println("İdeal hava!"); // Bu blok işləyəcək
        } else {
            System.out.println("Soyuq hava");
        }
    }
}
Məqsəd: Dəyişən elanı, if-else quruluşu və şərti məntiqi izah edir.
3. Try-Catch (Xəta idarəetmə)
import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        try {
            System.out.print("Birinci ədəd: ");
            int a = scanner.nextInt();
            System.out.print("İkinci ədəd: ");
            int b = scanner.nextInt();
            System.out.println("Nəticə: " + (a / b));
        } catch (ArithmeticException e) {
            System.err.println("Xəta: Sıfıra bölmə!"); // Əgər b=0 olarsa
        } catch (Exception e) {
            System.err.println("Gözlənilməyən xəta!");
        } finally {
            scanner.close(); // Resursu sərbəst burax
        }
    }
}
Məqsəd: Xəta idarəetmə mexanizmi (try-catch-finally), istifadəçi daxil etməsi.
